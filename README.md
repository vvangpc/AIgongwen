# 公文写作智能助手 v7.0 (Multi-Agent & Human-in-the-Loop)

这是一款基于 Python Streamlit 构建的轻量化但却拥有**商用级架构**的公文写作智能辅助 Web 平台。系统利用兼容 OpenAI 格式的大语言模型 API 接口，通过多名个性化设定的 Agent 在后台的互相推敲博弈，打破单纯的“一问一答”模式，为您提供极度缜密、贴合体制内行文规范的公文框架与润色定稿。

---

## 🌟 核心独家特性

1. **双轨 Agent 推敲系统 (Context Injection)**
   - 后台兵分两路：“主笔起草 Agent” 与 “审核挑刺 Agent”。
   - 根据您的调节轮数，他们会围绕您的初稿/需求进行激烈交锋、相互驳回与修改。
   - 核心防偏科逻辑（Context Drift Protection）确保了无论 AI 后台如何循环讨论，“用户的最初需求”始终被强制灌输为第一顺位指令，让每一轮生成都切中要害。

2. **热拔插定制提示词 (Popover UI)**
   - 点击侧边栏对应 Agent 名称，即刻弹出精美的轻量级小气泡 (Popover) 无缝更改人物设定。
   - 所有五名 Agent：框架主笔、框架处长、润色分析员、润色总监、排版大师，均可随时随您意愿灵活塑造风格。修改后直接实时生效。

3. **人机协同洗稿 (Human-in-the-loop 人在回路)**
   - 在润色的最后出稿前，系统绝不会越俎代庖。AI 吵架得到的润色方案将被拉出呈现到一个可编辑区。
   - 用户化身最高话事人，可强行在这组方案上划掉不想要的、补充想要加的（譬如要求全部改为排比句）。确认按下按钮后排版大师再忠实执行。

4. **打字机流式出稿 & 全方位断电保护 (State Safelock)**
   - 采用大模型 Streaming 技术进行打字机渲染，无需长时间等待转圈，极度舒适！
   - 加入了 Session State 缓存锚定和 `history.json` 本地固化机制。就算浏览器刷新、电脑重启，您的成品不仅依然原封不动坚挺在主界面，所有生平历史还将被永久记入第三页“档案室”。

## 🛠️ 安装环境与运行

本项目**不依赖任何复杂的环境甚至额外的数据库**。

**安装核心依赖库：**
```bash
pip install streamlit openai
```
或者运行：
```bash
pip install -r requirements.txt
```

**一键启动服务：**
```bash
python -m streamlit run app.py
```

服务启动后，系统会自动在您的默认游览器中打开交互平台，一切就这么简单。去体验公文 AI 新境界吧！
